# Bienvenido a este cuadernillo

Este cuadernillo es un recurso educativo diseñado para ayudarte a aprender y practicar habilidades de programación junto a mí. Aquí encontrarás una serie de ejercicios, ejemplos y explicaciones en el código que te guiarán a través de diferentes conceptos y técnicas de programación.

> [!NOTE]
> Este repositorio se irá actualizando conforme yo aprenda nuevos lenguajes de programación y nuevas técnicas, así que no dudes en volver a visitarlo para ver las novedades y seguir aprendiendo junto a mí.
